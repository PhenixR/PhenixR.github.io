Markdown 是一种适用于网络的书写语言，让文档更容易读、写和改动。Markdown的语法种类很少，只对应Html的一小部分。相对来说Html是一种发布的格式，而Markdown是一种书写格式，Markdown的格式语法只涵盖文本可以覆盖的范围，在某些方面更佳灵巧易用。

详细语法教程请移步Markdown 语法说明，这里只做简单的介绍。
